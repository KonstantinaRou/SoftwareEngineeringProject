%{color:orange}*Μη λειτουργικές απαιτήσεις*%

1.Το σύστημα θα πρέπει να έχει γρήγορη απόκριση σε περίπτωση αστοχίας. *_(Διαθεσιμότητα)_*
2.Η ταχύτητα απόκρισης του συστήματος θα πρέπει να είναι υψηλή ώστε να μπορεί να αυξηθεί η αποδοτικότητα.*_(Απόδοση)_*
3.Το σύστημα θα πρέπει να προστατεύει τους χρήστες από πιθανή έκθεση των στοιχείων τους σε μη εξουσιοδοτημένους χρήστες.*_(Ακεραιότητα)_*
4.Το λογισμικό θα πρέπει να προσφέρει την κατάλληλη συντηρησιμότητα ώστε να μπορεί να τροποποιηθεί και να διορθωθούν τυχόν σφάλματα με ευκολία. *_(συντηρησιμότητα)_* 
5.Το σύστημα θα είναι αξιόπιστο ως προς την λειτουργία του και την διάρκεια στο χρόνο. *_(Αξιοπιστία)_*
6.Το σύστημα θα μπορεί να ανακάμπτει μετά από πιθανή εσφαλμένη χρήση. *_(Ευρωστία)_*
7.Το λογισμικό του συστήματος θα είναι εύκολο προς εντοπισμό σφαλμάτων.*_(Ελεγξιμότητα)_*
8.Το σύστημα θα πρέπει να είναι εύχρηστο, εύκολο στην εκμάθηση του και φιλικό προς τον χρήστη.*_(Ευχρηστία)_*
9.Το σύστημα θα μπορεί να χρησιμοποιηθεί από χρήστες χωρίς ιδιαίτερες γνώσεις χρήσης ηλεκτρονικού υπολογιστή.*_(Ευχρηστία)_*
